本项目实现了论文“PUL-ABE: An Efficient and Quantum-Resistant CP-ABE With Policy Update in Cloud Storage”中的方案，其中的陷门生成和 SamplePre 机制是按照论文“Trapdoors for Lattices: Simpler, Tighter, Faster, Smaller”中的方法进行实现的。

This project implements the scheme proposed in the paper “PUL-ABE: An Efficient and Quantum-Resistant CP-ABE With Policy Update in Cloud Storage.”The trapdoor generation and the SamplePre mechanism are implemented following the methods described in “Trapdoors for Lattices: Simpler, Tighter, Faster, Smaller.”
