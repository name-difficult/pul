本项目实现了论文“PUL-ABE: An Efficient and Quantum-Resistant CP-ABE With Policy Update in Cloud Storage”中的方案，其中的陷门生成和 SamplePre 机制是按照论文“Trapdoors for Lattices: Simpler, Tighter, Faster, Smaller”中的方法进行实现的。为了实现该论文中的方法，项目中所使用的参数 q 并非质数，而是 2 的幂。此外，为了实现逆元素的计算，所有需要取逆的元素在随机生成时都增加了一个奇数的限制，即确保生成的随机数为奇数。

This project implements the scheme presented in the paper “PUL-ABE: An Efficient and Quantum-Resistant CP-ABE With Policy Update in Cloud Storage”, with the trapdoor generation and SamplePre mechanism following the methods described in “Trapdoors for Lattices: Simpler, Tighter, Faster, Smaller”. To implement the methods from this paper, the parameter q used in the project is not a prime number but a power of 2. Additionally, in order to compute inverses, all elements that require inversion are randomly generated with an added constraint that they must be odd, ensuring that the generated random numbers are odd.
